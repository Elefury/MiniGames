# Инструкция по развертыванию приложения

1. Скачайте архив проекта;
2. Распакуйте архив;
3. Откройте Unity Hub, загрузите проект из диска, выбрав распакованный проект MiniGames; 
4. Забилдив проект, откройте сцену SampleScene.unity, для этого откройте раздел Project, найдите внутри Assets папку Scenes, выберите её и дважды нажмите на SampleScene.unity, чтобы открыть сцену с игрой.
5. Можно играть. В редакторе встроен XR Device Simulator, можете играть через него, но если у вас есть VR-устройство, то лучше через него. :)
